T20 Score Predictor

T20 Score Predictor model will predict the total score at the end of innings by taking some factors into consideration such as batting team, bowling team, city, current score, wickets out, runs scored in last five overs etc. There are six processes involved in building this model. They are Data cleaning, EDA (Exploratory Data Analysis), Feature Engineering, Modelling, Website and Deployment.

Technologies Used : Machine Learning Algorithms like Linear Regression & Random Forest, Python and its packages such as Streamlit, Pickle, Pandas, NumPy, YAML, tqdm.

Driver Drowsiness Detection

Driver drowsiness detection is a car safety technology which helps prevent accidents caused by the driver getting drowsy. This system continuously assesses the driver's eyes and alerts him with alarms if the system detects that the driver closes his eyes very often. A webcam is required for this project for the system to monitor the driver's eyes regularly.

Technologies Used : Python, OpenCV, NumPy, Dlib, imutils python packages.
